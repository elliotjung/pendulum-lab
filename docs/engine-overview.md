# Engine Overview

Pendulum Lab is a framework-free, dependency-free TypeScript engine for nonlinear
pendulum dynamics, chaos analysis, and reproducible numerical experiments. Every
layer is unit-tested (954 tests across 144 files, synced from `reports/vitest-results.json`) and the build, typecheck, and Playwright
specs are green.

## Layers

| Layer | Module | What it provides |
|---|---|---|
| Numerics | `src/physics/integrators.ts`, `src/physics/adaptive.ts`, `src/physics/stiff.ts` | Twelve integrators from explicit Euler to a stiff TR-BDF2, plus adaptive control, embedded error estimates, and Richardson extrapolation |
| Physics | `src/physics/{double,triple,nPendulum,driven,spring}.ts` | Double, triple, generalized N-pendulum, driven/damped-driven, and elastic spring systems |
| Events | `src/physics/events.ts` | Bisection-located zero-crossings (Poincaré sections, period detection) |
| Chaos | `src/chaos/` | Lyapunov exponent (max + spectrum), Kaplan-Yorke dimension, SALI, FLI, Poincaré sampler, bifurcation sweep |
| Visualization | `src/viz/` | Pure-canvas, colorblind-safe renderers for energy, Lyapunov, Poincaré, bifurcation, and gradient traces |
| Performance | `src/workers/chaos.worker.ts`, `src/runtime/ChaosClient.ts` | Off-thread chaos compute over a typed message protocol, with a main-thread fallback |
| Validation | `src/validation/referenceSuite.ts` | Cross-validation of every integrator (order, energy, agreement) |
| Reproducibility | `src/research/reproPackage.ts` | Self-contained, hash-stamped run manifests that re-verify to the bit |

bounded-but-larger energy drift by design; TR-BDF2's drift reflects intentional
L-stable dissipation. See `docs/numerics.md` for the honesty notes per method.

## Chaos diagnostics (representative results)

| Quantity | Damped-driven preset (chaotic) | Harmonic oscillator (regular) |
|---|---|---|
| Maximal Lyapunov exponent (Benettin) | ≈ 0.10 | ≈ 0.000 |
| SALI | decays to ~1e-6 (exponential) | ≈ √2 (bounded) |
| FLI | ≈ 12 | ≈ 0 |

The conservative double pendulum's full Lyapunov spectrum reproduces the
Hamiltonian symplectic pairing (λ₁ ≈ −λ₄, sum ≈ 0) — a property that is not
hard-coded but falls out of a correct variational flow.

## Reproducibility

`npm run export:repro` writes verifiable run manifests. Because the integration
is fully deterministic, re-running a manifest reproduces the final state with
Δstate = 0, and the input content hash detects any tampering. Example: a double
pendulum integrated with `gbs` conserves energy to 1.3e-12 and verifies exactly.
